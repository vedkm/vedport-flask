FoliageFixer

Final Year Project

January 2023 to April 2023

For our final year project, my group (Funky - Ved Mahadeo, Ojore Kanneh, Hari Murti-Baran) knew we wanted to do something with machine learning. Our supervisor suggested plant disease classification for tomatoes and we really liked the idea. This was our first time working with machine learning but we were excited to take on the challenge.

Project Overview

The project consisted of an Android app, a Python/Flask backend and 2 neural network models: the Classification model and the Segmentation Model. The segmentation model isolates the diseased region of the leaf. The classification model takes the diseased region as input and classifies it as one of 10 diseases. Training the models required using 3rd party datasets as well as our own data. For segmentation, we chose a semantic segmentation approach which required manually labelling each pixel.

For a full analysis of the method, please check out the Project Report.

Tech used

  • Python/Flask
  • Heroku PostGres
  • PyTorch
  • OpenCV
  • Java
  • Android Studio
