WePala - Intern - May 2023 to August 2023

In 2023, I worked as an intern at WePala. WePala is an IT services company that specializes in helping entrepreneurs realise their technological vision.

Here, I was part of the WeOS team which is currently working on their new product: WeOS Cloud. As companies scramble to integrate ChatGPT in their services, WeOS aims to provide the unique service of streamlining ChatBot creation. In 5 minutes one can have a chatbot meticulously trained on company data.

My Contributions

  • Worked on a Golang back end project that mapped an internal database schema to external API schemas to allow GraphQL queries to pull data from multiple distributed sources using a universal language.
  • Extended the project by building a GraphQL API microservice for interacting with the Hubspot CRM, which involved writing a custom resolver in Golang to interpret the plain text queries and using the Echo framework to handle http requests.
  • Wrote 18 unit tests before implementing features according to the TDD framework, resulting in 75% code coverage.
  • Wrote and automated 10 Cypress/Cucumber End to End UI tests in JavaScript by following the BDD framework.

Tech used:

  • The Go Programming Language
  • JavaScript
  • Cypress
  • GraphQL
  • Github Actions

Key Takeaways

  • Automated End to end UI testing can be very useful for large front end applications that maybe difficult to manually test. It also ensures that things don't break after a new deploy.
